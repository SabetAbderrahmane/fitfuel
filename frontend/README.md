# FitFuel Frontend

Frontend scaffold for FitFuel AI.

## Stack
- Next.js 15 App Router
- TypeScript (strict)
- Tailwind CSS v4
- shadcn/ui
- TanStack Query v5
- React Hook Form + Zod
- Recharts
- Framer Motion
